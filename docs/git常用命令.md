
## git 常用命令
### 下拉git项目
$ git pull

### 下载git文件
$ git clone +git路径

### 切换git分支
$ git checkout +分支

### 选择要提交的git项目
$ git add .

### git提交人姓名
$ git commit -m "XXX"

### 上传git项目
$ git push origin +要上传的分支

### 暂存文件
$ git stash

### git flow工作流
- 更新develop，切换到自己的分支，右击develop，合并develop到自己的分支
- 切换到develop，将自己的分支合并到develop
